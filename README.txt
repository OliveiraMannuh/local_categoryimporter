Plugin Importar Categorias

---------------------------------------------------------

Plugin desenvolvido para importar categorias.

Compatibilidade: funciona a partir da versão 4.0 até 4.1.15

Modelo de arquivo de importação no formato csv:
name,description,idnumber,category_path

Campos obrigatórios para a importação do arquivo: name e category_path

Para fazer a importação acessar a administrção ao site, menu plugin, Plugins locais, Importar Categorias.

Nessa tela deve ser importado o arquivo em formato csv.
